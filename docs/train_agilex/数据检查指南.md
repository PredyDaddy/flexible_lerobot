# Agilex 数据集检查指南

本文档提供了 Agilex 机器人数据集的完整检查清单，确保录制的数据可以正常用于训练。

## 快速检查命令

```bash
# 查看数据集基本信息
cat ~/.cache/huggingface/lerobot/<your_username>/<dataset_name>/meta/info.json

# 可视化数据集
lerobot-dataset-viz --repo-id <your_username>/<dataset_name> --episode-index 0 --save 1 --output-dir ./viz_output
```

## 一、元数据一致性

检查 `meta/info.json` 文件：

| 检查项 | 期望值 | 说明 |
|--------|--------|------|
| `codebase_version` | `v3.0` | 版本兼容性 |
| `robot_type` | `agilex` | 机器人类型 |
| `fps` | `30` | 帧率 |
| `observation.state` shape | `[14]` | 双臂各7维 |
| `action` shape | `[14]` | 双臂各7维 |
| 相机数量 | 3 | left, right, front |
| 相机分辨率 | `640x480` | 与录制配置一致 |
| `features` 中的 dtype | `float32` / `int64` | 与实际数据匹配 |

**检查脚本：**
```python
import json
from pathlib import Path

repo = "your_username/agilex_dataset1"
root = Path.home() / ".cache/huggingface/lerobot" / repo

with open(root / "meta/info.json") as f:
    info = json.load(f)

print(f"版本: {info['codebase_version']}")
print(f"机器人: {info.get('robot_type', 'N/A')}")
print(f"FPS: {info['fps']}")
print(f"总帧数: {info['total_frames']}")
print(f"总episodes: {info['total_episodes']}")
print(f"Action维度: {info['features']['action']['shape']}")
print(f"State维度: {info['features']['observation.state']['shape']}")
```

## 二、统计信息完整性

检查 `meta/stats.json` 文件，确保所有 feature 都有完整的统计信息。

**必需的统计字段：**
- `min`, `max`, `mean`, `std`, `count`
- `q01`, `q10`, `q50`, `q90`, `q99` (分位数)

**图像特征要求：**
- shape 为 `(3, 1, 1)` (每通道一个值)
- 值在 `[0, 1]` 范围内

**检查脚本：**
```python
import json
from pathlib import Path

repo = "your_username/agilex_dataset1"
root = Path.home() / ".cache/huggingface/lerobot" / repo

with open(root / "meta/info.json") as f:
    info = json.load(f)
with open(root / "meta/stats.json") as f:
    stats = json.load(f)

required_keys = {'min', 'max', 'mean', 'std', 'count', 'q01', 'q10', 'q50', 'q90', 'q99'}

for feat in info['features']:
    if feat in stats:
        missing = required_keys - set(stats[feat].keys())
        if missing:
            print(f"⚠️ {feat}: 缺少 {missing}")
        else:
            print(f"✓ {feat}: 统计完整")
    else:
        print(f"❌ {feat}: 无统计信息")
```

## 三、帧数与时长

| 检查项 | 期望值 | 说明 |
|--------|--------|------|
| 每条轨迹帧数 | ~450帧 | 15秒 × 30fps |
| 时间戳间隔 | 0.0333秒 | 1/30 fps |
| 间隔标准差 | ~0 | 无丢帧/抖动 |

**检查脚本：**
```python
import pandas as pd
import numpy as np
from pathlib import Path

repo = "your_username/agilex_dataset1"
root = Path.home() / ".cache/huggingface/lerobot" / repo

# 读取 episodes 信息
episodes = pd.read_parquet(root / "meta/episodes/chunk-000/file-000.parquet")
data = pd.read_parquet(root / "data/chunk-000/file-000.parquet")

print("=== Episode 帧数 ===")
for i in range(len(episodes)):
    length = episodes.iloc[i]['length']
    duration = length / 30.0
    print(f"Episode {i}: {length} 帧 = {duration:.2f} 秒")

print("\n=== 时间戳间隔检查 ===")
expected_interval = 1.0 / 30.0

for ep_idx in range(len(episodes)):
    ep_data = data[data['episode_index'] == ep_idx]
    timestamps = ep_data['timestamp'].values
    intervals = np.diff(timestamps)

    abnormal = np.where(intervals > expected_interval * 2)[0]
    if len(abnormal) > 0:
        print(f"⚠️ Episode {ep_idx}: 发现 {len(abnormal)} 个异常间隔")
    else:
        print(f"✓ Episode {ep_idx}: 时间戳稳定")
```

## 四、动作/状态合法性

| 检查项 | 期望 | 说明 |
|--------|------|------|
| NaN 数量 | 0 | 无非法值 |
| Inf 数量 | 0 | 无无穷值 |
| 关节范围 | ±π 弧度内 | 安全范围 |
| 全零帧 | 0 | 无静止帧 |
| 最长连续静止 | <60帧 | 约2秒内 |

**检查脚本：**
```python
import pandas as pd
import numpy as np
from pathlib import Path

repo = "your_username/agilex_dataset1"
root = Path.home() / ".cache/huggingface/lerobot" / repo
data = pd.read_parquet(root / "data/chunk-000/file-000.parquet")

actions = np.array(data['action'].tolist())
states = np.array(data['observation.state'].tolist())

print("=== Action 检查 ===")
print(f"Shape: {actions.shape}")
print(f"NaN 数量: {np.isnan(actions).sum()}")
print(f"Inf 数量: {np.isinf(actions).sum()}")
print(f"范围: [{actions.min():.4f}, {actions.max():.4f}]")

# 全零帧检查
zero_frames = np.all(actions == 0, axis=1).sum()
print(f"全零帧: {zero_frames}")

# 连续静止检查
action_diff = np.diff(actions, axis=0)
static_frames = np.all(np.abs(action_diff) < 1e-6, axis=1)
max_consecutive = 0
current = 0
for is_static in static_frames:
    if is_static:
        current += 1
        max_consecutive = max(max_consecutive, current)
    else:
        current = 0
print(f"最长连续静止: {max_consecutive} 帧")

# 各维度范围
print("\n=== 各关节范围 ===")
names = ['L肩pan', 'L肩lift', 'L肩roll', 'L肘', 'L腕pitch', 'L腕roll', 'L夹爪',
         'R肩pan', 'R肩lift', 'R肩roll', 'R肘', 'R腕pitch', 'R腕roll', 'R夹爪']
for i, name in enumerate(names):
    col = actions[:, i]
    print(f"{name:10s}: [{col.min():7.3f}, {col.max():7.3f}]")
```

## 六、视频文件完整性

**检查脚本：**
```python
import subprocess
from pathlib import Path

repo = "your_username/agilex_dataset1"
root = Path.home() / ".cache/huggingface/lerobot" / repo

cameras = ['camera_left', 'camera_right', 'camera_front']

for cam in cameras:
    video = root / f"videos/observation.images.{cam}/chunk-000/file-000.mp4"

    # 检查可解码
    result = subprocess.run(['ffprobe', '-v', 'error', str(video)], capture_output=True)
    if result.returncode == 0:
        print(f"✓ {cam}: 可解码")
    else:
        print(f"❌ {cam}: 解码失败")

    # 获取帧数
    result = subprocess.run([
        'ffprobe', '-v', 'error', '-select_streams', 'v:0',
        '-show_entries', 'stream=nb_frames', '-of', 'csv=p=0', str(video)
    ], capture_output=True, text=True)
    print(f"   帧数: {result.stdout.strip()}")
```

## 七、任务标注

检查 `meta/tasks.parquet`：

```python
import pandas as pd
from pathlib import Path

repo = "your_username/agilex_dataset1"
root = Path.home() / ".cache/huggingface/lerobot" / repo

tasks = pd.read_parquet(root / "meta/tasks.parquet")
print("任务列表:")
print(tasks)

# 检查是否为占位符
for idx, row in tasks.iterrows():
    task_desc = list(row.keys())[0] if isinstance(row, pd.Series) else str(row)
    if "your task" in task_desc.lower() or task_desc == "":
        print(f"⚠️ 任务描述需要修改: {task_desc}")
```

**注意事项：**
- 任务描述应为实际任务内容（如 "抓取红色方块放入盒子"）
- 不要使用占位符文本
- 所有 episode 的 task_index 应一致（single_task 场景）

## 八、轨迹完整性

检查 episode 是否正常结束（最后几帧动作变化趋于平稳）：

```python
import pandas as pd
import numpy as np
from pathlib import Path

repo = "your_username/agilex_dataset1"
root = Path.home() / ".cache/huggingface/lerobot" / repo
data = pd.read_parquet(root / "data/chunk-000/file-000.parquet")
actions = np.array(data['action'].tolist())

print("=== 轨迹完整性检查 ===")
for ep_idx in range(5):
    ep_mask = data['episode_index'] == ep_idx
    ep_actions = actions[ep_mask]

    # 最后10帧 vs 整体平均变化
    last_10_diff = np.abs(np.diff(ep_actions[-10:], axis=0)).mean()
    overall_diff = np.abs(np.diff(ep_actions, axis=0)).mean()

    if last_10_diff < overall_diff * 0.5:
        print(f"✓ Episode {ep_idx}: 正常结束")
    else:
        print(f"⚠️ Episode {ep_idx}: 可能中途截断 (最后变化={last_10_diff:.4f}, 平均={overall_diff:.4f})")
```

## 十、训练配置匹配

确保以下配置与数据集一致：

| 配置项 | 数据集值 | 训练配置 |
|--------|----------|----------|
| 观测键名 | `observation.state`, `observation.images.*` | 策略配置中的 input_features |
| 动作键名 | `action` | 策略配置中的 output_features |
| 维度 | 14 | 策略配置中的 action_dim |
| FPS | 30 | delta_timestamps 计算基准 |

**delta_timestamps 检查：**
如果使用多帧上下文，确保 delta_timestamps 是 1/fps 的整数倍：

```python
fps = 30
delta_timestamps = [-0.1, 0.0]  # 示例

for dt in delta_timestamps:
    delta_frames = dt * fps
    if abs(delta_frames - round(delta_frames)) > 1e-4:
        print(f"⚠️ delta_timestamp {dt} 不是 1/{fps} 的整数倍")
    else:
        print(f"✓ delta_timestamp {dt} = {int(round(delta_frames))} 帧")
```

## 一键检查脚本

将以下脚本保存为 `check_dataset.py`：

```python
#!/usr/bin/env python3
"""Agilex 数据集一键检查脚本"""

import json
import subprocess
import numpy as np
import pandas as pd
from pathlib import Path

def check_dataset(repo: str):
    root = Path.home() / ".cache/huggingface/lerobot" / repo

    print(f"检查数据集: {repo}")
    print("=" * 50)

    # 1. 元数据
    print("\n[1/9] 元数据检查")
    with open(root / "meta/info.json") as f:
        info = json.load(f)
    print(f"  版本: {info['codebase_version']}")
    print(f"  FPS: {info['fps']}")
    print(f"  总帧数: {info['total_frames']}")
    print(f"  Episodes: {info['total_episodes']}")

    # 2. Stats 完整性
    print("\n[2/9] Stats 完整性")
    with open(root / "meta/stats.json") as f:
        stats = json.load(f)
    required = {'min', 'max', 'mean', 'std', 'count', 'q01', 'q10', 'q50', 'q90', 'q99'}
    all_ok = True
    for feat in info['features']:
        if feat in stats:
            missing = required - set(stats[feat].keys())
            if missing:
                print(f"  ⚠️ {feat}: 缺少 {missing}")
                all_ok = False
        else:
            print(f"  ❌ {feat}: 无统计")
            all_ok = False
    if all_ok:
        print("  ✓ 所有特征统计完整")

    # 3. 数据加载
    print("\n[3/9] 数据加载")
    data = pd.read_parquet(root / "data/chunk-000/file-000.parquet")
    print(f"  总行数: {len(data)}")

    # 4. NaN/Inf 检查
    print("\n[4/9] NaN/Inf 检查")
    actions = np.array(data['action'].tolist())
    nan_count = np.isnan(actions).sum()
    inf_count = np.isinf(actions).sum()
    if nan_count == 0 and inf_count == 0:
        print("  ✓ 无 NaN/Inf")
    else:
        print(f"  ❌ NaN: {nan_count}, Inf: {inf_count}")

    # 5. 时间戳稳定性
    print("\n[5/9] 时间戳稳定性")
    episodes = pd.read_parquet(root / "meta/episodes/chunk-000/file-000.parquet")
    all_stable = True
    for ep_idx in range(len(episodes)):
        ep_data = data[data['episode_index'] == ep_idx]
        intervals = np.diff(ep_data['timestamp'].values)
        if intervals.max() > 0.05:  # 超过 50ms
            print(f"  ⚠️ Episode {ep_idx}: 间隔不稳定")
            all_stable = False
    if all_stable:
        print("  ✓ 所有 episode 时间戳稳定")

    # 6. Frame Index 连续性
    print("\n[6/9] Frame Index 连续性")
    all_continuous = True
    for ep in range(len(episodes)):
        ep_data = data[data['episode_index'] == ep]
        if not np.array_equal(ep_data['frame_index'].values, np.arange(len(ep_data))):
            print(f"  ⚠️ Episode {ep}: 不连续")
            all_continuous = False
    if all_continuous:
        print("  ✓ 所有 episode frame_index 连续")

    # 7. 视频可解码
    print("\n[7/9] 视频可解码")
    for cam in ['camera_left', 'camera_right', 'camera_front']:
        video = root / f"videos/observation.images.{cam}/chunk-000/file-000.mp4"
        result = subprocess.run(['ffprobe', '-v', 'error', str(video)], capture_output=True)
        if result.returncode == 0:
            print(f"  ✓ {cam}")
        else:
            print(f"  ❌ {cam}")

    # 8. 轨迹完整性
    print("\n[8/9] 轨迹完整性")
    for ep_idx in range(len(episodes)):
        ep_actions = actions[data['episode_index'] == ep_idx]
        last_diff = np.abs(np.diff(ep_actions[-10:], axis=0)).mean()
        overall_diff = np.abs(np.diff(ep_actions, axis=0)).mean()
        if last_diff >= overall_diff * 0.5:
            print(f"  ⚠️ Episode {ep_idx}: 可能中途截断")
    print("  检查完成")

    # 9. 任务描述
    print("\n[9/9] 任务描述")
    tasks = pd.read_parquet(root / "meta/tasks.parquet")
    print(f"  {tasks.to_string()}")

    print("\n" + "=" * 50)
    print("检查完成!")

if __name__ == "__main__":
    import sys
    repo = sys.argv[1] if len(sys.argv) > 1 else "your_username/agilex_dataset1"
    check_dataset(repo)
```

**使用方法：**
```bash
python check_dataset.py your_username/agilex_dataset1
```

## 常见问题

### Q1: 任务描述是占位符怎么办？

目前 LeRobot 没有直接修改 tasks.parquet 的工具，需要重新录制或手动修改：

```python
import pandas as pd
from pathlib import Path

repo = "your_username/agilex_dataset1"
root = Path.home() / ".cache/huggingface/lerobot" / repo

# 创建新的 tasks DataFrame
tasks = pd.DataFrame({"task_index": [0]}, index=["抓取红色方块放入盒子"])
tasks.to_parquet(root / "meta/tasks.parquet")
```

### Q2: Episode 中途截断怎么办？

建议删除该 episode 并重新录制，或者在训练时排除该 episode。

### Q3: 数据量太少怎么办？

5 条数据仅作实验参考，建议：
- 至少录制 20-50 条高质量轨迹
- 覆盖不同的初始位置和角度
- 包含一些扰动恢复的轨迹
