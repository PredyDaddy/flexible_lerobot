# 服务器使用方法
## 服务器
```bash
python agilex_scripts/agilex_async_infer_single_cc_vertical.py server     --host 0.0.0.0 --port 8080 --fps 60
```

## 客户端
```bash
python agilex_scripts/agilex_async_infer_single_cc_vertical.py client --checkpoint /mnt/data2/cqy/workspace/my_lerobot/outputs/act_agilex_left_box/checkpoints/030000/pretrained_model --checkpoint-on-server --arm left --fps 60 --binary-gripper --server-address 10.0.0.30:8080 --actions-per-chunk 100 --chunk-size-threshold 0.3
```

# 本机推理
```bash
python agilex_scripts/agilex_async_infer_single_cc_vertical.py server \
      --host 127.0.0.1 --port 8080 --fps 60
```

```bash
python agilex_scripts/agilex_async_infer_single_cc_vertical.py client \
      --checkpoint outputs/act_agilex_left_box/checkpoints/030000/pretrained_model \
      --arm left --fps 60 --binary-gripper \
      --server-address 127.0.0.1:8080 \
      --actions-per-chunk 60 --chunk-size-threshold 0.3
```
