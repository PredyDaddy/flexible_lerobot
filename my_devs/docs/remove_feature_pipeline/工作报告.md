## 4. 架构说明

## 4.1 核心流程

1. 读取 arm 计划（left/right）：
   - left: 保留 `cam_high + cam_left_wrist`，切片 `[0:7]`
   - right: 保留 `cam_high + cam_right_wrist`，切片 `[7:14]`
2. 调用 LeRobot 现有 `remove_feature` 删除不需要腕部相机。
3. 对所有 `data/chunk-*/file-*.parquet` 重写：
   - `action` -> 7D
   - `observation.state` -> 7D
4. 更新 `meta/info.json` 中 shape（及可切片 names）。
5. 重算 `meta/stats.json`（action/state）并应用 `std_floor`。
6. 执行校验：
   - 相机组合正确
   - action/state shape 为 7
   - 无 NaN/Inf

## 4.2 入口

- CLI 入口：`my_devs/remove_feature_pipeline/single_arm_pipeline.py`
- 支持参数：`--source-root --source-repo-id --target-repo-id --arm --work-root --std-floor --dry-run --overwrite`

3. 修复：在 [core.py](/home/agilex/cqy/flexible_lerobot/my_devs/remove_feature_pipeline/core.py) 中将
