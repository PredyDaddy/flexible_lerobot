## 机器人端联调

默认先做 Phase 4 联调，保留 `--dry-consume true`，只消费异步队列并统计指标，不真实下发动作。

```bash
cd /home/agilex/cqy/flexible_lerobot

bash my_devs/train/groot/agilex/async/run_groot_remote_async_robot_client.sh \
  --server-host 10.1.26.37 \
  --server-port 5560 \
  --task "Grab the brown box and put it inside the orange one." \
  --robot-type agilex \
  --backend pytorch \
  --control-mode command_master \
  --control-arm right \
  --fps 30 \
  --low-watermark 1 \
  --chunk-size 2 \
  --response-timeout-s 0.75 \
  --reset-on-connect true \
  --dry-consume true \
  --run-time-s 60 \
  --log-level INFO
```

## 机器人端受控真机 Bringup

Phase 6 需要显式开启 `--actuate true`。默认仍然是安全关闭，不传这个参数不会下发动作。

首轮 bringup 约束：

- 显式写 `--control-arm right`，不要再用 `auto`
- 必须给正数 `--run-time-s`
- 首轮建议短时运行
- 如需保守一点，保留默认 `--actuation-warmup-steps 5`
- 默认带 `--max-action-age-ms 1000`，超过年龄阈值的预测动作会被替换成 hold

```bash
cd /home/agilex/cqy/flexible_lerobot

bash my_devs/train/groot/agilex/async/run_groot_remote_async_robot_client.sh \
  --server-host 10.1.26.37 \
  --server-port 5560 \
  --task "Grab the brown box and put it inside the orange one." \
  --robot-type agilex \
  --backend pytorch \
  --control-mode command_master \
  --control-arm right \
  --fps 30 \
  --low-watermark 1 \
  --chunk-size 2 \
  --response-timeout-s 0.75 \
  --reset-on-connect true \
  --dry-consume false \
  --actuate true \
  --actuation-warmup-steps 5 \
  --max-action-age-ms 1000 \
  --run-time-s 10 \
  --log-level INFO
```
