# 训练脚本使用日志与说明

目录：`my_devs/train/act/so101`

## 1. 脚本列表

- `train_smoke.sh`：快速管线验证
- `train_full.sh`：长时训练模板

## 2. 最常用命令

1. 先预演（推荐）

```bash
cd /home/agilex/cqy/flexible_lerobot/my_devs/train/act/so101
DRY_RUN=1 ./train_smoke.sh
```

2. 跑 smoke test

```bash
cd /home/agilex/cqy/flexible_lerobot/my_devs/train/act/so101
./train_smoke.sh
```

3. 跑 full train

```bash
cd /home/agilex/cqy/flexible_lerobot/my_devs/train/act/so101
./train_full.sh
```

## 3. 常见参数覆盖（环境变量）

```bash
# 指定数据集
DATASET_ROOT=/home/agilex/cqy/flexible_lerobot/datasets/lerobot_datasets/local__web_collection_demo \
DATASET_REPO_ID=local__web_collection_demo \
./train_smoke.sh

# 指定 GPU 与步数
CUDA_VISIBLE_DEVICES=0 POLICY_DEVICE=cuda STEPS=2000 ./train_full.sh

# 开启评估（仅在评估环境准备好时）
EVAL_FREQ=10000 ./train_full.sh
```
