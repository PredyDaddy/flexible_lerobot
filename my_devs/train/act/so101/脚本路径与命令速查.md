# SO101 ACT 脚本路径与命令速查

目录：`my_devs/train/act/so101`

本目录专门存放 **SO101** 的 ACT 训练、推理、验证脚本。

后续如果接入其他机器人，建议保持同样结构，例如：

- `my_devs/train/act/so101`
- `my_devs/train/act/xxx_robot`

## 1. 建议工作目录

最省事的方式：

```bash
cd /data/cqy_workspace/flexible_lerobot/my_devs/train/act/so101
```

如果你在仓库根目录下，也可以直接用完整相对路径执行。

## 3. 推理脚本

### 3.1 纯推理 dry-run

```bash

conda run -n lerobot_flex python my_devs/train/act/so101/run_act_infer.py --dry-run true
```

### 3.2 纯推理

```bash
cd /data/cqy_workspace/flexible_lerobot
conda run -n lerobot_flex python my_devs/train/act/so101/run_act_infer.py \
  --robot-port /dev/ttyACM0 \

  --wrist-cam-index 6 \
  --task "Put the block in the bin" \
  --run-time-s 120
```

## 6. 上机前建议重点确认

1. `POLICY_PATH` 是否指向 ACT checkpoint 的 `pretrained_model` 目录。
2. `ROBOT_PORT`、`TOP_CAM_INDEX`、`WRIST_CAM_INDEX` 是否和当前机器一致。
3. `CALIB_DIR` 是否对应当前 SO101 follower 标定目录。
4. 验证录制时 `DATASET_REPO_ID` 是否以 `eval_` 开头。
5. 如果 ACT 重规划太慢，优先尝试 `POLICY_N_ACTION_STEPS=16` 或 `8`。
